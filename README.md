# Student Management System

Backend REST API built using Spring Boot to manage student records.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL / H2

## Features
- Create, Read, Update, Delete students
- REST APIs
- Layered architecture

## API Endpoints
- GET /api/students
- GET /api/students/{id}
- POST /api/students
- PUT /api/students/{id}
- DELETE /api/students/{id}

## How to Run
1. Clone repo
2. Run Spring Boot app
